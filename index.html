The engagement

Not a training.
An operating system upgrade.

Week 1-2: Equip

Get tools in place (GH Copilot / Cursor / Claude Code). Hands-on with real work, not demos. Tower's people start building with AI on day one.

Month 1: Amplify

2-3 key people, coached weekly on their actual projects. Learn to specify, validate, and iterate with AI. They become the internal experts.

Month 2-3: Multiply

Those people train others. Leadership coaching on AI strategy. The capability spreads organically, not through top-down mandates.

Ongoing: Evolve

Models change quarterly. New capabilities unlock new possibilities. A strategic partner keeps Tower current and ahead.

"Building Gen AI is not the same as using it.

Equip people, not processes. The people scale everything else.

Getting started

The tool stack: practical options

GitHub Copilot

VS Code extension with agent mode. Access to Opus 4.6, Gemini, GPT. Choose the model per task. Enterprise plan with admin controls and audit logs. Likely easiest security approval path.

~$19/user/month

Claude Code CLI

The full harness. Autonomous agents that plan, build, test, and iterate on real codebases for hours. Terminal-based, maximum control. The tool behind the 100K-line C compiler demo.

Usage-based pricing

Cursor

VS Code fork with AI native to the IDE. Multi-model support (Claude, GPT, Gemini). Familiar editor experience, low onboarding friction. Already in use by some Tower teams.

~$20/user/month

All three access the same frontier models (Opus 4.6, Gemini, GPT).
The difference is interface, not intelligence. Start where security allows.
